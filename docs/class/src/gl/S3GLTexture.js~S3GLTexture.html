    S3GLTexture

    Extends:

    * → S3GLTexture

    WebGL描画用のテクスチャクラス。 S3Textureを拡張し、WebGL用のGLTexture管理、GL用データ取得（getGLData）、破棄などを担います。 画像データをGPUのテクスチャへ変換し、GLSLシェーダへのuniformバインドなどに利用します。
