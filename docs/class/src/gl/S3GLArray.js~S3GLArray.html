    S3GLArray

    WebGL用の配列（バッファ）を生成・管理するクラス。（immutable） 各種型（S3Vector, S3Matrix, 数値配列等）をWebGLバッファ（Float32Array/Int32Array）に変換し、 対応するGLSL型（vec3, mat4等）情報も保持します。
