    S3GLShader

    WebGLのシェーダー管理クラス。 頂点シェーダ／フラグメントシェーダのソースコード・型・GLオブジェクトを保持し、コンパイルや破棄、状態取得などの機能を提供します。 S3GLProgram 内部で利用され、単体では直接使わないことが多い設計です。
