    S3GLSystem

    Extends:

    S3System → S3GLSystem

    WebGLレンダリングシステムを管理するクラス。 シェーダー、テクスチャ、バッファオブジェクトの生成・管理、および描画制御を担当。 WebGLの初期化やプログラムのセットアップ、シーンの描画などの処理を含む。
