    S3System

    Direct Subclass:

    S3GLSystem

    3DCGシステム全体を管理するクラス

    3DCGのための座標変換やシーン管理、基本的な生成処理・ユーティリティ関数などをまとめて提供します。 頂点やメッシュ、マテリアルなど各種オブジェクトのファクトリ機能も持ちます。
